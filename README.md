# Learning Tracker (Django Project)

## Description
A Learning Tracker web application built using Django to help users track daily learning progress, manage courses, and maintain consistency.

## Features
- User account management
- Course tracking system
- Progress monitoring
- Organized learning structure

## Tech Stack
- Python
- Django
- SQLite

## My Contribution
Developed the complete backend using Django, created models, views, and handled project structure and functionality.

## How to Run
1. Install dependencies
2. Run: python manage.py runserver
3. Open in browser: http://127.0.0.1:8000/
